### BMI Calculator

A simple Python program that calculates Body Mass Index (BMI) based on user input.

```python
weight = float(input("Weight (kg): "))
height = float(input("Height (m): "))

bmi = weight / (height ** 2)

print("BMI:", round(bmi, 2))
